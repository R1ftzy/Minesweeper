# Minesweeper

A tui minesweeper game built with Go and Bubbletea.

## Requirements

- [Go](https://go.dev/dl/) installed on your machine

## Usage

Clone the repo and run:

```
git clone https://github.com/YOUR_USERNAME/minesweeper
cd minesweeper
go run .
```

## Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move cursor |
| Enter / Space | Reveal cell |
| F / P | Place or remove flag |
| R | Restart |
| Q | Quit |

## Difficulty

Select difficulty at the start with arrow keys and Enter.

| Mode | Grid | Bombs |
|------|------|-------|
| Easy | 5×5 | 5 |
| Medium | 9×9 | 10 |
| Hard | 16×16 | 40 |
